A plugin for allowing the control over what your users can see in the admin section of WordPress

== Description ==

Simply add the RS User Access. Select a user from the dropdown and start assigning the partss they can access.

== Installation ==

1. Upload `rs-user-access` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
